u = \(x^\alpha y^{1-\alpha}\) s.t. 2x+3y \(\leq\) 12 . Find the optimal solution.

  • lagrange ,\(Z= x^\alpha y^{1-\alpha} + \lambda (12-2x-3y)\)
  • first order conditions are
  • \(\frac{\partial Z}{\partial x} = \alpha x^{\alpha -1} y^{1-\alpha} - \lambda 2= 0\)
  • \(\frac{\partial Z}{\partial y} = x^{\alpha} \alpha y^{- \alpha } - \lambda 3 = 0\)
  • \(\frac{\partial Z}{\partial \lambda } = 12-2x-3y =0\)
  • from above equations we get
  • \(\frac {\alpha x^{\alpha - 1} y^{1-\alpha}} {\alpha x^{\alpha} y^{- \alpha }} = \frac {\lambda 2}{\lambda 3}\)
  • \(\implies \frac{y}{x} = \frac{2}{3 }\)
  • \(\implies y=\frac {2x}{ 3}\)
  • putting the value of y in \(12-2x-3y=0\) we get
  • \(12-2x-3(\frac{2x}{3 })=0\)
  • \(12-4x=0\)
  • \(x=3\)
  • \(y=\frac{2x}{3}=2\)
  • hence the optimal consumption bundle
  • \((x*,y*)=(2,3)\)

\(U= x^{0.2} y^{0.8}\) s.t \(3x+4y\leq 12\)

  • from First order conditions we get
  • \(\frac{0.2 x^{-0.8} y^{0.8}}{0.8x^{0.2}y^{-0.2}}=\frac{3\lambda}{4\lambda}\)
  • \(\implies \frac{y}{x}=\frac {3}{4}\)
  • \(\implies y= \frac{3x}{4}\)
  • substituting \(y=\frac{3x}{4}\) in \(12-3x-4y= 12\) we get,
  • \(12-3x-4(\frac{3x}{4})=0\)
  • \(\implies 12-6x=0\)
  • \(\implies x=2\)
  • \(\implies y= \frac{3x}{4}=\frac{3}{2}\)
  • hence,
  • \((x*,y*)= (2,\frac{3}{2})\)
