Linear Utility Function

  • Linear Utility functions are in the form of ax+by
  • \(mrs_{x,y}\) is constant ( independent of quantity of other good)
  • Both goods are normal goods ( positive coefficients , \(a,b>0\))
  • if \(mrs_{x,y}=\frac{P_x}{P_y}\) then All points on budget line including corner points are optimal solution .
  • Corner solution means The good with higher “marginal utility of per unit spend” is only consumed.In other words consumer substitutes all his spending on the good where she gets higher utility .
  • if \(mrs_{x,y}<\frac{P_x}{P_y}\) then \((x*,y*)=(0,\frac{M}{P_y})\)
  • if \(mrs_{x,y}>\frac{P_x}{P_y}\) then \((x*,y*)=(\frac{M}{P_x},0)\)

\(u=2x+3y\) \(4x+6y=24\) \((x*,y*)= any \ point \ on \ budget \ line\)
\(u=3x+5y\) \(6x+9y=108\) \((x*,y*)=(0,12)\)
\(u=\min[x,y]+max[x,y]\) \(2x+3y=60\) \((x*,y*)=(30,0)\)

\(u=x+y\) s.t. \(2x+3y=12\)

\(mrs_{x,y}=\frac{\frac{\partial u}{\partial x}}{\frac{\partial u}{\partial y}}=1\) \(\frac{P_x}{P_y}=\frac{2}{3}\) as \(mrs_{x,y}>\frac{P_x}{P_y}\) \((x*,y*)=(\frac{M}{P_x},0)=(\frac{12}{2},0)=(6,0)\)

\(u=2x+3y\) s.t. \(4x+6y=24\)

\(mrs_{x,y}=\frac{\frac{\partial u}{\partial x}}{\frac{\partial u}{\partial y}}=\frac{2}{3}\) \(\frac{P_x}{P_y}=\frac{4}{6}=\frac{2}{3}\) as \(mrs_{x,y}=\frac{P_x}{P_y}\) there are infinite solutions ( any point on the budget line) \(x*=0\) to \(\frac{M}{P_x}\) and \(y*=\frac{M-P_x x}{P_y}\)

\(u=3x+5y\) s.t. \(6x+9y=108\)

\(mrs_{x,y}=\frac{\frac{\partial u}{\partial x}}{\frac{\partial u}{\partial y}}=\frac{3}{5}\) \(\frac{P_x}{P_y}=\frac{6}{9}=\frac{2}{3}\) as \(mrs_{x,y}<\frac{P_x}{P_y}\) \((x*,y*)=(0,\frac{M}{P_y})=(0,\frac{108}{9})=(0,12)\)

It is not always necessary that You will find Linear Utility function in ax+by form. Some times utility functions are monotonic transformation of other utility function or represented in another way. (MRS is some constant)

For example \(u=\min[x,y]+max[x,y]\)

Here if \(x>y\) then \(u=x+y\) if \(x<y\) then \(u=x+y\) if \(x=y\) then \(u=x+y\)

So No matter what the quantities are \(u=x+y\)

So solving \(u=x+y\) with a budget constraint like \(2x+3y=60\) we get \((x*,y*)=(30,0)\)

  • If both goods are perfect substitute (a=b) ,then good with lower price is “only” consumed . If \(P_x<P_y\) then \((\frac{M}{P_x},0)\) is optimal solution. \(u=2x+2y\) s.t. \(3x+4y=24\) then \((x*,y*)=(8,0)\)
  • If both goods have same price \((P_x=P_y)\) then , The good with bigger coefficient in utility function is consumed . \(u=x+2y\) s.t. \(3x+3y=12\) then \((x*,y*)=(0,4)\)

A case where there are more then 2 goods

if \(u=2x_1+3x_2+x_3\) s.t \(4x+5y+3z=120\)

Same logic still applies 1

Simply find the good with higher marginal utility per unit spent.

Here \(\frac{\frac{\partial u}{\partial x}}{P_x}=\frac{2}{4}=\frac{1}{2}\) \(\frac{\frac{\partial u}{\partial y}}{P_y}=\frac{3}{5}\) \(\frac{\frac{\partial u}{\partial z}}{P_z} = \frac{1}{3}\)

as \(\frac{\frac{\partial u}{\partial z}}{P_z} < \frac{\frac{\partial u}{\partial y}}{P_y} <\frac{\frac{\partial u}{\partial x}}{P_x}\)

Optimal solution is where the consumer spends all her income on z and consumes 0 unit of x and y each.

\((x*,y*,z*)=(0,0,\frac{M}{P_z})=(0,0,\frac{120}{3})=(0,0,40)\)
